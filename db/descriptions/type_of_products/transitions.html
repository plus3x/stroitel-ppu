Переходы

    Как правило, не всегда трубопровод состоит из труб одного диаметра. Иногда необходимо соединить трубы различного диаметра. Для этого и были сконструированы переходы. Переходы представляют собой элементы, в которых диаметры сечений с обоих краев различные. Кроме того, подобные элементы используются для соединения ППУ трубопровода с обычными трубами.

    Переходы бывают различных видов. Так, например, существуют элементы с двумя концами состыковки с разным диаметром, но есть и разветвленные элементы, с одной стороны которых можно подсоединить одну трубу, а с другой две.

    Для прокладки по современным технологиям переходы покрыты слоем пенополиуретана с целью минимизации потерь тепла, поскольку используются для транспортировки веществ с довольно высокой температурой. В случае прокладки под землей целесообразно использовать переходы с полиэтиленовым внешним покрытием, над землей — с оцинкованным.

    Иногда переходы могут использоваться для изменения давления транспортируемого вещества из трубы одного в трубу другого диаметра.
