Рядовые элементы

Фасонный рядовой элемент представляет собой трубу с кабельным выводом, который может применяться для вывода лишнего количества какого-либо вещества из системы либо же, наоборот, для подпитки сети.

Конструкция рядового элемента довольно проста — это та же труба, посередине которой находится кабельный вывод. Ось вывода перпендикулярна оси основной проводящей трубы, а диаметр значительно меньше нее.

Рядовые элементы имеют место во всех видах труб, и предизолированные — не исключение. Они покрыты специальным теплоизолирующим слоем — пенополиуретаном, который снижает потери тепла. Внешний защитный кожух может быть выполнен из полиэтилена для подземных трубопроводов или из оцинкованной трубы для наземных конструкций.
